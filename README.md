# 🌡️ Thermo Guardian

A real-time temperature monitoring system developed using **Arduino Uno** and **DHT11/LM35 Temperature Sensor**. The project continuously measures ambient temperature and displays the readings in real time, providing a simple, accurate, and cost-effective solution for temperature monitoring.

---

## 📖 Overview

Temperature monitoring is essential in various domains such as healthcare, industries, laboratories, smart homes, and environmental monitoring. **Thermo Guardian** automates the process of temperature detection by eliminating the need for manual measurements and providing instant temperature updates.

---

## ✨ Features

* 🌡️ Real-time temperature monitoring
* 📟 Live temperature display
* ⚡ Accurate and reliable temperature readings
* 🔄 Continuous monitoring without manual intervention
* 💰 Low-cost and easy-to-build solution
* 🔧 Expandable for IoT and alert-based systems

* DHT11 / LM35 Temperature Sensor
* OLED Display / LCD Display
* Breadboard
* Jumper Wires

---

## ⚙️ Working Principle

1. The DHT11/LM35 sensor measures the surrounding temperature.
2. The Arduino Uno reads the sensor data.
3. The data is processed by the microcontroller.
4. The temperature is displayed on the screen in real time.
5. The system continuously monitors and updates the readings.
